# 🛒 Supermarket Sales Performance Dashboard 📊

## 🎯 Project Objective
This analysis aims to monitor supermarket sales trends, evaluate branch performance, and understand customer satisfaction levels across different product lines.

## 🔑 Key Insights from Dashboard
* **Total Sales:** Reached over 307K with a gross income of 15.38K.
* **Customer Demographics:** Comparison between Member and Normal customer types.
* **Branch Analysis:** Visualizing sales distribution across branches (A, B, and C).
* **Product Performance:** Identifying top product lines like Food & Beverages and Sports & Travel.

## 🛠️ Tech Stack
* **Power BI:** Data Visualization & Dashboarding.
* **Data Source:** Supermarket sales historical record.
